## Module Breakdown

### bin/orca.js
- Entry point
- Command parsing (commander)
- Route to appropriate command handler
- Handle unknown commands

### src/commands/kickstart-python.js
- Main kickstart logic
- Prerequisites checking
- Repository cloning
- Virtual environment setup
- Dependency installation
- Server startup
- Process management

### src/utils/index.js
- `commandExists()` - Check if command is available
- `getPythonCommand()` - Find python3 or python
- `getVenvPaths()` - Get OS-specific paths
- `runCommand()` - Execute command with output
- `runCommandSilent()` - Execute command silently
- `spawnBackground()` - Spawn background process
- `waitForPort()` - Wait for port availability
- `print.*` - Colored output helpers

- Pip: `orca_env/bin/pip`
